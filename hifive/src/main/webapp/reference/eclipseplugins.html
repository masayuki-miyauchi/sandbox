コード補完プラグイン

hifiveコード補完プラグインをインストールすると以下に示すコード補完が出来るようになります。
ただし、hifiveのコード規約、命名規約を順守している必要があります。補完対象の詳細に関してはチュートリアルを参照。

  • コントローラ
  • ロジック
  • イベントコンテキスト
  • イニシャライザコンテキスト

例

var selectorController = {
  __name: 'SelectorController',
  'input[type=button] click': function(context) {
    this. //このthisはコントローラ(selectorController)なのでコントローラの補完を出す
    context. //このcontextはイベントコンテキストなのでイベントコンテキストの補完を出す
  }
};
  • インストール後の設定方法
    下記リンクからオプションファイルをダウンロードします。(右クリック -> 名前を付けて保存)

    オプションファイルのリンク : hifiveCodeAssist設定ファイル

    ダウンロードしたオプションファイルはeclipseプロジェクトの任意の場所に保存してください。
    プロジェクトを右クリック -> プロパティ -> hifiveCodeAssist
    からダウンロードしたオプションファイルを指定してください。

    以上の設定を行うことでhifive用のコード補完が使用できるようになります。

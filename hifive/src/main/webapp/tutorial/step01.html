hifiveのダウンロード

まず初めにこちらからzipをダウンロードしましょう。

downloadページから必要なモジュールを選んでダウンロードすることもできますが、 このチュートリアルでは全機能を含んだjsファイルを使用します。

まずはzipファイルをダウンロードしてください。

ファイル構成

ダウンロードしたzipファイルを解凍すると、以下のような構成になっています。{version}はダウンロードしたバージョンによって異なります。

  • h5-{version}.dev.js
    • 開発用のhifiveファイルです。開発時用にログを出力したり、様々な開発支援機能を含んでいます。
    • チュートリアルでh5.jsと表記されているものについてはこちらを使用してください。
  • h5-{version}.js
    • リリース用のhifiveファイルです。ログ出力や、開発支援機能は含みません。hifiveを使ったプロジェクトを公開するときに使用してください。
  • h5-{version}.css
    • hifiveで使用するcssファイルです。
    • チュートリアルでh5.cssと表記されているものについてはこちらを使用してください。
  • ejs-1.0.h5mod.js
    • hifiveで使用するテンプレート機能用のjsファイルです。テンプレート機能についてはチュートリアル05.ビュー操作で説明します。

jQueryのダウンロード

hifiveはjQueryとjQuery.blockUIを使用します。以下のファイルをダウンロードしてください。

  • jQuery-1.7.1以上
  • jQuery.blockUI

使用方法

以下のようにして、html内でダウンロードしたcssファイル、jsファイルを読み込みます。 

		<!doctype html>
        <html>
            <head>
                <meta charset="UTF-8">
                <link type="text/css" rel="stylesheet" href="lib/h5/h5.css">
                <script src="lib/jquery/jquery.js"></script>
                <script src="lib/jquery/plugin/jquery.blockUI.js"></script>
                <script src="lib/h5/ejs-1.0.h5mod.js"></script>
                <script src="lib/h5/h5.js"></script>
            </head>
            <body>
                ...
            </body>
        </html>
 
 
準備ができたら、具体的なコードに進む前に、hifiveにおけるアプリケーション構造の考え方について学びましょう。
